Do you want to contribute?

Thank you so much for wanting to collaborate with us!!

The best way to help us is by spreading the word about the language. Using it, telling others about it, and enjoying it. We have already used it, and the experience has been great.

Although there is still a lot of work to do. Libraries, text editors, applications are missing... the list is endless.

Good news for computer scientists: a new toy to explore :-)

stxt.dev

Our language has infinite applications, but we believe its definition is unique and already finished.

We leave the possible applications or uses for you ;-)

As for this website, we believe it is a closed project, but it is far from perfect. Surely it can be improved in a thousand ways. Do you think a new chapter should be added to the book? Have you found any errors in the documentation? Would you improve any example? Would you suggest a different and more compact css?

We want this website to be a reference point for learning STxT.
